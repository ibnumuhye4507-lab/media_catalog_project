# рІерібрѕхрѕІрѕЏрІі рѕџрІ▓рІФ ріФрЅ│рѕјрїЇ рѕЏрѕхрЅ░рІ│рІ░рѕфрІФ (Islamic Media Catalog)

рІГрѕЁ рЇЋрѕ«рїђріГрЅх рІерЅ░рѕѕрІФрІЕ рібрѕхрѕІрѕЏрІі рѕўрїйрѕљрЇЇрЅхріЋ ріЦріЊ рЅфрІ▓рІ«рІјрЅйріЋ рЅарЅђрѕІрѕЅ рѕѕрѕЏрѕхрЅ░рІ│рІ░рѕГ ріЦріЊ рѕфрЇќрѕГрЅх рѕѕрѕЏрІЇрїБрЅх рІерЅ░рѕ░рѕФ рІерЇЊрІГрІўріЋ (Python) рѕХрЇЇрЅхрІїрѕГ ріљрІЇрЇб

##  рІІріЊ рІІріЊ рЅ░рїЇрЅБрѕФрЅх
* **рѕўрѕерїЃ рѕўрѕўрІЮрїѕрЅЦрЇА** ріарІ│рІ▓рѕх рѕўрїйрѕљрЇЇрЅхріЋ ріЦріЊ рЅфрІ▓рІ«рІјрЅйріЋ рЅа XML рЇІрІГрѕЇ рІЇрѕхрїЦ рѕўрѕўрІЮрїѕрЅЦрЇб
* **рІ│рѕйрЅдрѕГрІх рѕЏрѕўріЋрїерЅхрЇА** рІерЅ░рѕўрІўрїѕрЅАрЅхріЋ рѕўрѕерїЃрІјрЅй рІѕрІ░рѕџрІФрѕЮрѕГ ріарѕЏрѕГріЏ HTML рѕфрЇќрѕГрЅх рѕўрЅђрІерѕГрЇб
* **рІўрѕўріЊрІі рѕіріЋрі«рЅйрЇА** рѕўрїйрѕљрЇЇрЅхріЋ ріе Google BooksрЇБ рЅфрІ▓рІ«рІјрЅйріЋ рІ░рїЇрѕъ ріе YouTube рїІрѕГ рЅарЅђрїЦрЅ│ рѕЏрїѕріЊріўрЅхрЇб
* **рЅђрѕІрѕЇ ріарїарЅЃрЅђрѕЮрЇА** рѕЏріЋріЏрІЇрѕЮ рѕ░рІЇ рЅарЅђрѕІрѕЅ ріЦріЋрІ▓рїарЅђрѕЮрЅарЅх рЅаріарѕЏрѕГріЏ рЅ░рѕГрѕџріЊрѕЇ рІерЅ░рІўрїІрїђрЇб

##  рІерЅ░рїарЅђрѕЎрЅБрЅИрІЇ рЅ┤ріГріќрѕјрїѓрІјрЅй
1. **Python:** рѕўрѕерїЃрІЇріЋ рѕѕрѕЏрѕхрЅ░рІ│рІ░рѕГ (Backend logic)рЇб
2. **XML:** рѕўрѕерїЃрІјрЅйріЋ рЅарЅІрѕџріљрЅх рѕѕрѕЏрѕхрЅђрѕўрїЦ (Database)рЇб
3. **XSLT:** рІе XML рѕўрѕерїЃріЋ рІѕрІ░ HTML рѕѕрѕўрЅђрІерѕГ (Transformation)рЇб
4. **HTML/CSS:** рІерѕџрІФрѕЮрѕГ рІ│рѕйрЅдрѕГрІх рѕѕрѕўрЇЇрїарѕГрЇб

## ­ЪЊќ ріЦріЋрІ┤рЅх рѕўрїарЅђрѕЮ рІГрЅ╗рѕІрѕЇ?
1. рЇЋрѕ«рїЇрѕФрѕЎріЋ рІФрѕхріљрѕ▒рЇА `python src/main.py`
2. рЅарѕџрѕўрїБрІЇ ріарѕЏрѕФрїГ рѕўрѕ░рѕерЅх рѕўрѕерїЃ рІФрѕхрїѕрЅА рІѕрІГрѕЮ рѕфрЇќрѕГрЅх рІФрІЇрїАрЇб
3. рІерІѕрїБрІЇріЋ рѕфрЇќрѕГрЅх рѕѕрѕЏрІерЅх `report.html` рЇІрІГрѕЇріЋ рЅарЅЦрѕФрІЇрІўрѕГ рІГріГрЇѕрЅ▒рЇб
